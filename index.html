Website Purpose & Goals

1,The website is about showing the relationship between the production of Australia tuna aquaculture and the value of import and export commodities value.
2, The purpose is to promote tuna aquaculture, improve the development of Australian commodity economy.

Process of Data Cleaning

1, Dataset evaluation: understand the structure of data, manually clear missing values, data type conversion, outlier processing, etc
2, Data integration: If there are multiple data sources, it may be necessary to merge them into a single data set. This can be done by associating different data sets with common fields or keys.
3, Data validation: by using Google sheet to create charts to validate the results of the cleaned dataset to ensure the accuracy and consistency of the data.
